-- 6. Between, to show The BETWEEN operator is used in a WHERE clause to filter the result set within a certain range. The range must be separated by an AND keyword.For example, this query filters the result to only include shows between the years 2020 and 2025 (inclusive):
-- Or to show all table 2 different range but in same cell
SELECT * 
FROM shows
WHERE year
BETWEEN 2000 and 2025;  

-- 7 ORDER BY, to show The ORDER BY statement sorts rows of data in ascending or descending order. By default, this command sorts the data in ascending order.
-- mengurutkan baris data dalam urutan naik atau turun. Secara default, perintah ini mengurutkan data dalam urutan naik
-- Write code below ðŸ’–
SELECT name, genre, stream, year
FROM shows
ORDER BY year;

-- Tahun terbaru / Order from the most recent year
SELECT name, genre, stream, year
FROM shows
ORDER BY year DESC;
